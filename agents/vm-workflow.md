# VM workflow

This is a disposable VM/sandbox dedicated to autonomous AI work. Commit your own
work — when you finish a unit of work that leaves a git repository with
uncommitted changes, commit it yourself without being asked, rather than ending
your turn with changes left uncommitted. Write a real commit message describing
the change and follow the repository's commit conventions (including its
trailer/co-author and AI-disclosure policy). If `HEAD` is detached or on the
default branch, create a topic branch first. Don't push unless asked.
